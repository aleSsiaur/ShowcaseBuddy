# ShowcaseBuddy
ShowcaseBuddy 是一个面向设计学生的设计项目叙事包装与展示页生成 Skill。它帮助用户将零散项目材料转化为完整的项目展示包装方案，包括项目核心叙事、标题与简介、展示逻辑结构、多模型效果包装 Prompt Pack，以及网页展示页结构。与普通作品集建议不同，ShowcaseBuddy 聚焦“可展示、可生成、可落地”的表达流程，尤其支持根据已有设计作品生成适配主流生图模型的展示效果图 Prompt，如网页Hero、展陈场景、mockup、过程材料包装图等。MVP 聚焦网页展示页，帮助设计学生把“项目做完了”转化为“讲得清、看起来完整、能被展示”的作品集表达。
